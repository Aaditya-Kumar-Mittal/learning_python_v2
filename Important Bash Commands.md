# 🐍 Python Interpreter & File Commands (Git Bash)

## 📄 Create a New Python File

Create a Python file:

```bash
touch file.py
```

Example:

```bash
touch hello.py
```

---

## ▶️ Start the Python Interpreter

Launch the interactive Python shell:

```bash
python
```

You should see something similar to:

```text
Python 3.13.14 (main, ...)
>>>
```

You can now execute Python code interactively.

Example:

```python
>>> print("Hello, World!")
Hello, World!

>>> x = 10
>>> x * 2
20
```

---

## ❌ Exit the Python Interpreter

### Option 1 (Recommended)

```python
exit()
```

---

### Option 2

```python
quit()
```

---

### Option 3 (Keyboard Shortcut)

Press:

```text
Ctrl + D
```

---

## 💡 Quick Reference

| Task                     | Command         |
| ------------------------ | --------------- |
| Create a Python file     | `touch file.py` |
| Start Python interpreter | `python`        |
| Exit interpreter         | `exit()`        |
| Exit interpreter         | `quit()`        |
| Exit using keyboard      | `Ctrl + D`      |

---

## 📝 Example Workflow

```bash
touch hello.py

python
```

```python
>>> print("Hello, Aaditya!")
Hello, Aaditya!

>>> exit()
```
